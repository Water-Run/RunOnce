# 软件定义

`RunOnce`/`一次运行`是一个发布于微软商店的Windows桌面端应用, 使用`C#`+`WinUI3`技术栈.  
提供一个一次性编辑器, 主要用于执行大模型生成的一次性脚本.  
程序支持i18n: 简体中文,英文(其它系统).  
需要注册到Win11的右键菜单. 典型使用是, 右键--在此运行代码, 打开编辑器.  
软件仅适用于Win11, 使用Win11亚克力Mica, 遵循最标准的Win11 WinUI3的设计风格.  
程序支持多个语言. 需要加入权重打分机制(优化算法), 自动识别语言; 用户也可以进行手动设置.  
编辑器支持基础的语法高亮.  

选择语言:  

- 最可能  
- 次可能  
- 手动指定  

执行时, 也选择语言.  
设置项包括:  

- 风格: 跟随系统/浅色/深色  
- 显示语言: 跟随系统/中文/English
- 语言指令配置: 比如Python是`python`, lua是`lua`等
- 临时文件名前缀: 默认为`__RunOnceTMP__`  
- 执行前语言选择框: 始终显示/有语言高可信时不显示(默认)/仅无语言可信时显示
- 执行前确认: 开/关(默认关)

原理是: 在目录下生成一个临时文件, 并调用语言指令执行(兜底临时文件删除).
